## 🔍 **PROPRIÉTÉS DU MODÈLE UTILISÉES**

### **Sources de données**
| Section | Tables/Collections | Calculs |
|---------|-------------------|---------|
| **KPIs** | `sales_documents`, `customers` | Agrégations temps réel |
| **Pipeline** | `sales_documents` (status) | Filtres par état |
| **Performances** | `commercials`, `sales_metrics` | Comparaisons N-1 |
| **Alertes** | Toutes tables | Règles métier + seuils |

## 📌 **FONCTIONNALITÉS CLÉS**

1. **Vue 360°** : Synthèse complète activité commerciale
2. **Temps réel** : Actualisation automatique
3. **Drill-down** : Du global au détail en 1 clic
4. **Alertes intelligentes** : Priorisation critique/important
5. **Comparaisons** : vs objectifs, N-1, moyennes
6. **Actions directes** : Boutons contextuels
7. **Export** : Excel, PDF, programmation envois

## 🎯 **INDICATEURS MÉTIER**

- **CA et objectifs** : Suivi quotidien/mensuel
- **DSO** : Days Sales Outstanding (délai encaissement)
- **Pipeline** : Visibilité commandes en cours
- **Taux conversion** : Devis → Commandes
- **Performance équipes** : Ranking commerciaux
- **Géographie** : Analyse zones de vente
- **Produits** : Top ventes et progressions

## 🚀 **OPTIMISATIONS UX**

1. **Visualisations** : Graphiques, jauges, sparklines
2. **Codes couleur** : 🟢 OK, 🟡 Attention, 🔴 Critique
3. **Responsive** : Adapté écran/tablette
4. **Personnalisation** : Widgets configurables
5. **Alertes push** : Notifications temps réel
